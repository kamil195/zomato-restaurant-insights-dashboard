# Project Summary

## Business objective

The dashboard provides a high-level view of Zomato restaurant data so users can compare restaurant concentration, cuisine popularity, customer ratings, and service availability across locations.

## Dataset coverage

- 9,551 restaurants
- 250 cuisines
- 15 countries
- 141 cities
- Approximately 1 million votes
- Average rating of 2.89

## Dashboard components

### KPI cards

The main dashboard displays total restaurants, cuisines, countries, cities, votes, and average rating.

### Filters

- Country slicer
- Year selector

### Location analysis

The dashboard compares restaurant counts by country and city. India represents the majority of records, while New Delhi is the largest city in the dataset.

### Cuisine analysis

The leading cuisine categories include North Indian, Chinese, Fast Food, Mughlai, Italian, Bakery, Continental, and Cafe.

### Service availability

The report compares table-booking and online-delivery availability across restaurants.

## Recommended future improvements

- Add a geographic map visual
- Add tooltip pages for city and cuisine details
- Display percentages with service-availability totals
- Add rating and cost segmentation
- Document the data source and Power Query transformations
