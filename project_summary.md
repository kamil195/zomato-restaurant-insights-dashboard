# Project Summary

## Business objective

The dashboard provides a high-level view of Zomato restaurant data so users can compare restaurant concentration, cuisine popularity, customer ratings, and service availability across locations.

## Dataset coverage

- 9,551 restaurants
- 250 cuisines
- 15 countries
- 141 cities
- Approximately 1 million votes
- Average rating of 2.89

## Main findings

- India: 8,652 restaurants
- New Delhi: 5,473 restaurants
- North Indian cuisine: 3,960 restaurants
- Rating bucket 3–4: approximately 4.6K restaurants
- Table booking: about 1.16K Yes and 8.39K No
- Online delivery: about 2.45K Yes and 7.1K No

## Recommended improvements

- Replace the long country slicer with a searchable dropdown
- Use a more consistent color palette across report pages
- Add tooltip pages for city and cuisine details
- Display percentages beside booking and delivery totals
- Add a geographic map
- Add rating and cost segmentation
- Export a clean dashboard screenshot without the Power BI Desktop interface
