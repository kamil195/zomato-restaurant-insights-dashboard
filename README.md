# Zomato Restaurant Insights Dashboard

An interactive Power BI project analyzing restaurant coverage, cuisines, ratings, votes, table-booking availability, and online-delivery availability across multiple countries and cities.

## Dashboard overview

The dashboard summarizes 9,551 restaurants, 250 cuisines, 15 countries, 141 cities, approximately 1 million votes, and an overall average rating of 2.89.

## Main analysis areas

- Restaurant distribution by country and city
- Cuisine popularity and restaurant counts
- Average ratings by restaurant brand
- Table-booking availability
- Online-delivery availability
- Year and country filtering
- Supporting country and date dimension tables

## Key findings

- India dominates the dataset with 8,652 restaurants.
- New Delhi has the largest city-level presence with 5,473 restaurants.
- North Indian is the most common cuisine, appearing in 3,960 restaurants.
- The largest rating group is 3–4, containing roughly 4.6K restaurants.
- Approximately 1.16K restaurants support table booking, while 8.39K do not.
- Approximately 2.45K restaurants offer online delivery, while 7.1K do not.

## How to view the project

1. Download `Zomato_Insights_Dashboard.pbix` from the repository root.
2. Open it using Microsoft Power BI Desktop.
3. Use the country and year filters to explore the dashboard interactively.
4. Open `dashboard_overview.pdf` for a non-interactive overview.
5. Read `docs/project_summary.md` for the project summary and findings.
